# Mintaskbot

まぁコマンドを打つとランダムに時間帯ごとに返信してくれます。

# How to setting  

npm install discord.js

# 仕様書

## 概要

Bot名：気分転換Bot
目的：ユーザーが気軽に「短時間の気分転換」を提案してくれる

## 動作環境

Node.js 18以上
discord.js v14
動作対象：特定ギルド（サーバー）

## コマンド仕様

/kibun

説明：
現在の時間帯に合わせた気分転換の提案を返す

## 使用者

サーバー内の誰でも

返信形式：Ephemeral（本人にのみ表示される）

## 処理内容
ユーザーのIDを元に、最後に提案した内容を記憶
時間帯に応じたリストからランダムに提案
直前の提案は避けるように選択

## 時間帯分類
morning    05:00–09:59    背伸びをしてみる、窓を開けて朝日を取り込む
daytime    10:00–15:59    温かい飲み物を飲む、軽い散歩
evening    16:00–20:59    背筋を伸ばす、読書
night    21:00–04:59    目を休める、軽くストレッチ
